## Activation

```tsx
// apps/docs/app/layout.tsx
<html data-ui="liquid">...</html>
```

```css
/* apps/docs/app/global.css */
@import 'fumadocs-ui/css/liquid-tinted.css';
[data-ui="liquid"] { --font-sans: -apple-system, BlinkMacSystemFont, "SF Pro", "Segoe UI", Roboto, Helvetica, Arial, sans-serif; }
```

## Usage

- Components pick up tokens automatically via existing classes.
- Surfaces apply blur and gradients under Liquid activation.

## Testing

```bash
pnpm run --filter docs dev
BASE_URL=http://localhost:3000 pnpm --filter docs test:visual
```

## Extending

- Add custom tint accents by adjusting `--color-fd-primary`.
- Scope additional effects under `[data-ui="liquid"]`.
